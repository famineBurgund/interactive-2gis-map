# interactive-2gis-map
 Интерактивная карта наружной рекламы, созданная на основе 2gis API, предназначенная для администрирования броней наружной рекламы.
 Карта создавалась для конкурса КУБ 4.0, но мы, loxichi, в финал не прошли, а потому карта чут-чут не доделана.
 Для работы карты вам понадобиться 2GIS API, а также база данных, структура которой описана при помощи моделей Sequelize (добавил sql скрипты готовых данных).
 В целом всё...

 Для запуска карты нужно:
 1. Создать PostgreSQL БД и ввести строку подклчения в файл db.js.
 2. Создать ключ 2GIS, получить можно на https://dev.2gis.ru/api, ключ вводим в компонент Map2GIS.jsx.
 3. Запустить npm run dev в папке backend.
 4. Запустить npm start в папке frontend.
